## Obfuscation (Basic)

1. Access [https://obfuscator.io/](https://obfuscator.io/).
2. Change “String Array Encoding” to “Base64”.
3. Paste JavaScript code.
4. Click "Obfuscate".
5. Try running it on [https://jsconsole.com/](https://jsconsole.com/).
